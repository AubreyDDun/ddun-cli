# sao-vue-ts-template

> vue+ts工程初始化

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao vue-ts-template my-project
```

### From git

```bash
sao aubreyddun/sao-vue-ts-template my-project
```
